## Showcase Projects

### AI Safety and LLM Security

**[LLM Safety Evaluation](https://github.com/NAMEisNOTvailable/llm-safety-evaluation)**

Mandarin-English prompt-injection evaluation project for measuring refusal and compliance behaviour across models, languages, and attack styles.

- 1,500 matched Mandarin-English prompt pairs.
- Response files and scoring workflow for comparative safety analysis.
- Cross-lingual risk framing around complete refusal, partial compliance, and full compliance.

**[LLM Defend](https://github.com/NAMEisNOTvailable/llm-defend)**

Chinese-first prompt-injection dataset composer for building coverage-balanced adversarial prompts and reproducible LLM safety evaluation inputs.

- Deterministic generation pipeline with seeded composition.
- Deduplication workflow using similarity and hashing approaches.
- Audit sidecars, statistics reports, and data-provenance notes for reviewability.

### Security Assessment

**[GuardedIM Security Assessment](https://github.com/NAMEisNOTvailable/guardedim-security-assessment)**

Local security-assessment lab for a secure messaging prototype with Python client/server components and Go control-plane code.

- Reproducible protocol flaws with local PoC demos.
- Review coverage across RSA key exchange, AES-GCM message handling, file validation, and framing.
- Python and Go tests through GitHub Actions, plus mitigation-oriented documentation.

### Machine Learning Evaluation

**[CNN CIFAR-10 Classification](https://github.com/NAMEisNOTvailable/cnn-cifar10-classification)**

PyTorch image-classification project comparing an ImageNet-pretrained ResNet-18 fine-tuned on CIFAR-10 against an AlexNet-style CNN trained from scratch.

- Executed notebook with training logs, confusion matrices, and result plots.
- Script-based workflow with reusable `src/`, tests, and CI smoke checks.
- Clear caveat that the reported result is a selected single run, not a multi-seed benchmark.

**[Medical MLP Classification](https://github.com/NAMEisNOTvailable/mlp-medical-classification)**

TensorFlow tabular-classification project exploring MLP depth, SMOTE, and metric-aware evaluation on an imbalanced medical dataset.

- Validation-based model selection and threshold-aware reporting.
- SMOTE applied only to the training split.
- Reproducible script workflow with checksum-protected data loading, result files, tests, and notebook report view.
